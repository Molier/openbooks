Key UX Improvements

Summary of the most impactful changes

↕️

Vertical Scrolling Only

Eliminates horizontal scrolling by converting table to vertically-stacking cards. Users never lose context.

📖

Full Text Visibility

Book titles and author names are fully visible (up to 2 lines with smart truncation), improving "information scent."

📚

Smart Grouping

Same book from multiple servers becomes ONE card with "5 more sources" disclosure. Reduces visual clutter by 60-80%.

⚠️

Inline Error Handling

Parsing errors appear as specially-marked cards in the results list, not hidden in a separate view.

👆

Touch-Friendly Targets

All interactive elements meet 48dp minimum size for easy tapping. Download buttons are prominent.

📏

Consistent Formatting

File sizes always use standardized format: "351 KB", "1.5 MB", "Size unknown" – never raw numbers.
